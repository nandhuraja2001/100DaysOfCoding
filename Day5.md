## Statement
```
Write a program that takes three numbers and prints their sum. Every number is given on a separate line.
```
## Sample input
```
3
5
```
## Code
```
# Read the numbers b and h like this:
b = int(input())
h = int(input())
print(1/2*b*h)
# Print the result with print()
```
## Sample output
```
7.5
```
