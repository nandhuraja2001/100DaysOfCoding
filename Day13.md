## Statement
```
Given an integer, print "odd" if it's odd and print "even" otherwise.
```
## Sample input 1
```
5
```
## Sample input 2
```
6
```
## Code
```
# Read an integer:
# a = int(input())
# Print a value:
# print(a)
a = int(input())
if(a%2==0):
  print("even")
else:
  print("odd")
```
## Sample output 1
```
odd
```
## Sample output 2
```
even
```
